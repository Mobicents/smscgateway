Name
	smpp esme show

SYNOPSIS
	smpp esme show <esmeName>

DESCRIPTION
	This command is used to list all configured ESMEs.

PARAMETERS
	esmeName		- Name of the ESME to show.
	This parameter is optional. All ESMEs will be displayed in case of
	no esmeName is specified.
