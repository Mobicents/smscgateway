Name
	smsc set smdeliveryfailure_tpfailurecause

SYNOPSIS
	smsc set smdeliveryfailure_tpfailurecause <code> <permanent|temporary|clear>

DESCRIPTION
	This command is used to set the code of MAPErrorMessageSMDeliveryFailure
	with SMS-DELIVERY-REPORT to	needed values.
	- If it set to 'permanent' then SMSC will treat the error with <code> as permanent
	- If it set to 'temporary' then SMSC will treat the error with <code> as temporary
	- This value is set to 'clear' by default.

EXAMPLES
	smsc set smdeliveryfailure_tpfailurecause  <code> temporary

SEE ALSO
	smsc get smdeliveryfailure_tpfailurecause
